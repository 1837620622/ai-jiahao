## 这是什么？

「嘉豪」是 2025–2026 出圈的网络梗：从隔壁班黑卫衣、口罩、模仿 Alan Walker 打碟，到「用力过猛的青春显眼包」，再到后来的——

> **质疑嘉豪，理解嘉豪，成为嘉豪。**

本项目把这股劲接到 **AI 认知测评** 上：

1. **先自评段位**（四档难度不同）
2. **必做卷满分 100 + 附加彩蛋分**
3. **每题轮换四位陪考官 3D 形象**（特朗普 / 高市早苗 / 莫迪 / 马克龙）
4. **测完签发嘉豪人格**（称号与专属形象一一绑定，不是随机抽图）

适合：自测 AI 底子、朋友圈整活、团队团建、课程导入、面试热身。

## 功能一览

- **四段位**：新手上路 · 进阶玩家 · 高手过招 · 大神登顶  
- **计分清晰**：必做卷满分 100，附加题是彩蛋分，过线只看必做  
- **大题库**：1300+ 题，每次开考刷新抽取  
- **本场不重复**：同场题目 ID 唯一  
- **跨场防撞题**：`localStorage` 记住近期做过的题，尽量不重复  
- **即时判分 + 解析**  
- **10 种嘉豪人格 + 专属 3D 形象**  
- **四位陪考官轮值吐槽 / 加油**  
- **结果可分享**（复制文案 / 系统分享）

## 段位与计分

| 段位 | 必做 | 计分 | 附加 | 过线（必做） | 适合谁 |
|------|------|------|------|--------------|--------|
| 新手上路 | 20 题 | ×5 分 = **100** | 5×2（最多 +10） | ≥ 60 | 会用 ChatGPT 但说不清原理 |
| 进阶玩家 | 25 题 | ×4 分 = **100** | 8×1（最多 +8） | ≥ 70 | 经典配置，最接近参考卷 |
| 高手过招 | 25 题 | ×4 分 = **100** | 10×2（最多 +20） | ≥ 75 | 要把 Demo 做成能上线 |
| 大神登顶 | 25 题 | ×4 分 = **100** | 10×2（最多 +20） | ≥ 80 | 架构 / 对齐 / 推理服务 |

**规则说明：**

- **必做卷**：决定是否过线 + 人格主判定（按必做正确率）  
- **附加卷**：答对加分，答错不扣必做分；拉开稀有人格与徽章  
- **每次「开考 / 同配置再战」都会重新抽题**

## 陪考官同学

本场**不设监考，只请气氛组**（不改分）：

| 同学 | 定位 | 人设 |
|------|------|------|
| 特朗普同学 | 气势位 | 答对 Beautiful，答错 Sad |
| 高市早苗同学 | 严谨位 | 审题、概念、别被带节奏 |
| 莫迪同学 | 定力位 | 深呼吸，想清楚再选 |
| 马克龙同学 | 表达位 | 逻辑要干净，措辞要站得住 |

**每答一题换一位 3D 形象出镜**，左侧大图 + 气泡点评。

## 题库与防重复

- 题池按初级 / 中级 / 高级池组织，段位映射到不同池  
- 覆盖：LLM、Token、提示工程、RAG、Agent、评测、安全、推理工程等  
- `JIAHAO.buildPaper(tierId)`：Fisher–Yates 洗牌 + 无放回抽取  
- 本场 `id` 去重  
- `localStorage` 键 `jiahao_used_v1` 记录近题；设置页可「清空近题记录」

## 本地运行

```bash
git clone https://github.com/1837620622/ai-jiahao.git
cd ai-jiahao

# 任意静态服务器均可
```
